# Markdown to HTML

## Project structure

```
20231026 markdown to html pandoc
├─ docs_template.html
├─ example.html
├─ example.md
├─ md_to_html.bat
└─ README.md
```

## Usage

Using `md_to_html.bat` file convert `example.md` to `example.html`.

File `md_to_html.bat` uses `docs_template.html` - custom pandoc html template.
